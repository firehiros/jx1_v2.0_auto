NOTES:
===========================================

1. Color format: Use hex colors (#RRGGBB or #AARRGGBB)
2. Sizes: All sizes are in pixels unless specified
3. Booleans: Use Yes/No or checked/unchecked
4. Lists: Separate items with commas
5. Comments: Lines starting with # are comments
6. Placeholders: Use {0}, {1}, etc. for dynamic values

===========================================
HOW TO APPLY CHANGES:
===========================================

After editing this file, run:

   PowerShell:
   > cd launcher
   > .\apply_design.ps1

   Or manually rebuild:
   > cd launcher
   > dotnet build

Changes will be applied to MainWindow.xaml and code-behind.
